--[ 0 - Introduction

awesome is a highly configurable, next generation framework window manager for
X. It is very fast, extensible and licensed under the GNU GPLv2 license.

awesomewm one of the most configurable's window managers i've ever seen
it provides a lua api which you can use to do literally anything.

  - Keybindings
    - global.lua: Global keybindings
    - client.lua: Client keybindings like movings around and stuff
  - Modules
    - autostart.lua: autostart for applications
    - err_handler.lua error handler functions
    - notification.lua: naughty notification configuration.
    - rules.lua: window rules
    - signals.lua: awesomewm signals
    - utils.lua: helper functions
  - themes: awesomewm theme
  - widgets: statusbar widgets


--[ 1 - Installation

install the release version of awesomewm.
and clone the repo inside ~/.config/awesome

$ git clone https://github.com/itsnexn/awesomewm ~/.config/awesome

and restart your awesomewm.
